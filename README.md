# CrashAlert

## Overview
CrashAlert predicts accident probability for a given route using historical data.

## Features
- Route-based crash prediction
- Heatmap visualization
- Flask-based web interface

## Tech Stack
- Python
- Flask
- HTML/CSS/JS

## How to Run
1. Clone repo
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   python app.py

## Data Sources
- accidents.csv
- landslide_zones.csv
